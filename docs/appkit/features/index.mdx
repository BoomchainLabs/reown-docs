While AppKit offers a wide range of features that make it easy to build interactive and user friendly blockchain applications, certain features are limited to selected frameworks.

**The following table provides an overview of the features available in AppKit and the frameworks they are supported in.**

- ✅ - indicates the feature is supported and can be integrated.
- ❌ - indicates the feature is not supported and cannot be integrated.
