Send notifications through Reown Cloud's notification service.

## Sending Notifications

Use the Reown Cloud API to send notifications to your users:

```ts
// Send notification via Cloud API
const sendNotification = async (userId: string, message: string) => {
  const response = await fetch('https://api.reown.com/v1/notifications', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${projectId}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      userId,
      type: 'transaction',
      title: 'Transaction Update',
      message,
      data: {
        timestamp: Date.now()
      }
    })
  })
  
  return response.json()
}
```

## Notification Payload

| Field | Type | Description |
|-------|------|-------------|
| `userId` | string | Target user identifier |
| `type` | string | Notification type |
| `title` | string | Notification title |
| `message` | string | Notification content |
| `data` | object | Additional metadata |

## Delivery Options

- **Immediate**: Send notification immediately
- **Scheduled**: Schedule for future delivery
- **Batch**: Send to multiple users
- **Conditional**: Send based on user preferences

Visit [Reown Cloud](https://cloud.reown.com) to manage your notification sending quotas and delivery settings.
