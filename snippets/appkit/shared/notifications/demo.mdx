## Code Example

```ts
// Demo notification setup
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  features: {
    analytics: true,
    notifications: {
      enabled: true,
      demo: true // Enable demo mode
    }
  }
})

// Trigger demo notification
modal.notify({
  type: 'success',
  title: 'Demo Notification',
  message: 'This is a test notification from AppKit!'
})
```
