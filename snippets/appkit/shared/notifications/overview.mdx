AppKit Notifications provide real-time updates and alerts for wallet activities and dApp interactions.

## Features

- Real-time transaction notifications
- Wallet connection alerts
- Custom notification types
- Cross-platform delivery

## Configuration

Enable notifications in your AppKit configuration:

```ts
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  features: {
    analytics: true,
    notifications: true // Enable notifications
  }
})
```

## Notification Types

| Type | Description |
|------|-------------|
| `transaction` | Transaction status updates |
| `connection` | Wallet connection events |
| `network` | Network change notifications |
| `custom` | Custom application notifications |

Visit [Reown Cloud](https://cloud.reown.com) to configure notification settings and delivery preferences.
