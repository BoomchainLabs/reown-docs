You can configure custom networks by passing them to the `networks` array in your AppKit configuration.

```ts
import { createAppKit } from '@reown/appkit'
import { WagmiAdapter } from '@reown/appkit-adapter-wagmi'
import { defineChain } from '@reown/appkit/networks'

// Define custom network
const customNetwork = defineChain({
  id: 1234,
  caipNetworkId: 'eip155:1234',
  chainNamespace: 'eip155',
  name: 'Custom Network',
  nativeCurrency: {
    decimals: 18,
    name: 'Custom Token',
    symbol: 'CTK'
  },
  rpcUrls: {
    default: {
      http: ['https://rpc.custom-network.com']
    }
  },
  blockExplorers: {
    default: {
      name: 'Custom Explorer',
      url: 'https://explorer.custom-network.com'
    }
  }
})

const wagmiAdapter = new WagmiAdapter({
  networks: [customNetwork],
  projectId
})

const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [customNetwork],
  metadata,
  projectId
})
```

Custom networks allow you to connect to any EVM-compatible blockchain that isn't included in the default network list.
