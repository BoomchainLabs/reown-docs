Reown Cloud provides blockchain API services for enhanced dApp functionality.

## Configuration

Access blockchain APIs through your AppKit configuration:

```ts
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  features: {
    analytics: true,
    blockchainApi: true // Enable blockchain API features
  }
})
```

## Available APIs

- **Transaction History**: Retrieve user transaction history
- **Token Balances**: Get real-time token balances
- **NFT Collections**: Access NFT metadata and collections
- **Price Data**: Current and historical price information
- **Network Stats**: Blockchain network statistics

## Usage

Blockchain APIs are automatically integrated when enabled. Visit [Reown Cloud](https://cloud.reown.com) to configure API access and rate limits for your project.
