Reown Cloud Verify provides wallet and domain verification services.

## Configuration

Enable verification features in your AppKit configuration:

```ts
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  features: {
    analytics: true,
    verify: true // Enable verification features
  }
})
```

## Verification Types

- **Wallet Verification**: Verify wallet authenticity and reputation
- **Domain Verification**: Confirm dApp domain legitimacy
- **Transaction Verification**: Validate transaction authenticity
- **Smart Contract Verification**: Verify contract source code

## Benefits

- **Enhanced Security**: Protect users from malicious actors
- **Trust Indicators**: Display verification badges
- **Fraud Prevention**: Detect and prevent fraudulent activities
- **Compliance**: Meet regulatory requirements

Visit [Reown Cloud](https://cloud.reown.com) to configure verification settings and manage your verification status.
