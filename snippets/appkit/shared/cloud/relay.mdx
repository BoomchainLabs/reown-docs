Reown Cloud Relay provides secure communication infrastructure for wallet connections.

## Configuration

Relay services are automatically configured when using AppKit:

```ts
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  // Relay is automatically configured
})
```

## Features

- **Secure Communication**: End-to-end encrypted messaging
- **Cross-Platform**: Works across web, mobile, and desktop
- **High Availability**: Redundant infrastructure for reliability
- **Global Network**: Optimized for worldwide performance

## Benefits

- **Enhanced Privacy**: No data is stored or logged
- **Improved Performance**: Optimized routing and caching
- **Seamless Experience**: Transparent to end users
- **Scalable Infrastructure**: Handles high-volume applications

Relay services are included with your Reown Cloud project. Visit [Reown Cloud](https://cloud.reown.com) to monitor usage and configure settings.
