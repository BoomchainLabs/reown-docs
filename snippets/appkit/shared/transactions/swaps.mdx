Swap functionality enables users to exchange cryptocurrencies directly within your dApp.

## Configuration

Enable swap features in your AppKit configuration:

```ts
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  features: {
    analytics: true,
    swaps: true // Enable swap functionality
  }
})
```

## Opening Swap Modal

Trigger the swap interface programmatically:

```ts
// Open swap modal
modal.open({ view: 'Swap' })

// Open with pre-filled values
modal.open({
  view: 'Swap',
  arguments: {
    fromToken: 'ETH',
    toToken: 'USDC',
    amount: '1.0'
  }
})
```

## Supported Features

- **Multi-DEX Aggregation**: Best prices across multiple DEXs
- **Slippage Protection**: Configurable slippage tolerance
- **Gas Optimization**: Efficient transaction routing
- **Price Impact**: Real-time price impact calculation
- **Token Search**: Comprehensive token database

Swap functionality provides users with seamless token exchange capabilities without leaving your application.
