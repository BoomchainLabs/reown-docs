On-ramp functionality allows users to purchase cryptocurrency directly within your dApp.

## Configuration

Enable on-ramp features in your AppKit configuration:

```ts
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  features: {
    analytics: true,
    onramp: true // Enable on-ramp functionality
  }
})
```

## Opening On-Ramp

Trigger the on-ramp modal programmatically:

```ts
// Open on-ramp modal
modal.open({ view: 'OnRamp' })

// Open with specific parameters
modal.open({
  view: 'OnRamp',
  arguments: {
    currency: 'USD',
    amount: '100',
    cryptocurrency: 'ETH'
  }
})
```

## Supported Features

- **Multiple Providers**: Integration with various on-ramp providers
- **Currency Support**: Support for multiple fiat currencies
- **Cryptocurrency Options**: Purchase various cryptocurrencies
- **KYC Compliance**: Built-in compliance features
- **Payment Methods**: Credit card, bank transfer, and more

On-ramp functionality provides a seamless way for users to acquire cryptocurrency without leaving your application.
