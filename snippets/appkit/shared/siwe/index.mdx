Sign-In with Ethereum (SIWE) provides a standard way for users to authenticate with your dApp using their Ethereum wallet.

## Configuration

Enable SIWE in your AppKit configuration:

```ts
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  features: {
    analytics: true,
    siwe: true // Enable SIWE
  }
})
```

## Implementation

SIWE authentication is handled automatically by AppKit when enabled. Users will be prompted to sign a message after connecting their wallet.

## Benefits

- **Secure Authentication**: Cryptographic proof of wallet ownership
- **Standard Protocol**: Uses the EIP-4361 standard
- **User Control**: Users maintain full control of their authentication
- **Privacy**: No personal information required

Visit [Reown Cloud](https://cloud.reown.com) to configure SIWE settings for your project.
