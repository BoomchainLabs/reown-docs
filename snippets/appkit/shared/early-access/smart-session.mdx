Smart Sessions provide enhanced session management capabilities for AppKit.

## Configuration

Enable Smart Sessions in your AppKit configuration:

```ts
const modal = createAppKit({
  adapters: [wagmiAdapter],
  networks: [mainnet, arbitrum],
  metadata,
  projectId,
  features: {
    analytics: true,
    smartSessions: true // Enable Smart Sessions
  }
})
```

## Features

Smart Sessions provide:

- Extended session duration
- Cross-device session synchronization
- Enhanced security features
- Automatic session renewal

## Usage

Smart Sessions are automatically managed by AppKit when enabled. They provide a seamless user experience by maintaining wallet connections across browser sessions and devices.

Visit [Reown Cloud](https://cloud.reown.com) to configure Smart Session settings for your project.
